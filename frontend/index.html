    Steam Libraries

    The app scans these folders to find your installed games. Steam libraries are detected automatically from your installation. Add paths here for any drives or custom locations not picked up automatically. Removing a path will exclude it from future scans.

      NexusMods

      Your API key is used to look up mod info and download files. Get yours at nexusmods.com → account → API Keys. Premium members get direct downloads; free members can use the "Mod Manager Download" button on NexusMods to get an NXM link.

      Changes take effect after restarting the game.
